## ARcade
Platform Pembelajaran Interaktif Berbasis Augmented Reality dan AI yang Inklusif untuk Semua Gaya Belajar.

Project Link : https://ar-cade-fe.vercel.app

Technology :
- [Next JS](https://nextjs.org/).
- [TailwindCSS](https://tailwindcss.com/).

## Prerequisite
- Already install [nodejs](https://nodejs.org/).
- Already install [npm](https://www.npmjs.com/).

## Step by step installation
#### 1. Clone this repository
```
git clone https://github.com/Rekognisi-Garis-Keras/arcade-fe.git
```
or 
#### Download the zip file
<a href="https://github.com/Rekognisi-Garis-Keras/arcade-fe/archive/refs/heads/main.zip" target="_blank"><img src="https://github.com/0x1m4o/Industry-Project/blob/main/public/img/image.png" width="400" alt="Download ZIP"></a>

#### 2. Go to directory 
```
cd arcade-fe
```

#### 3. Install dependency with npm
```
npm install
```

#### 4. Start the server
```
npm run dev
```
